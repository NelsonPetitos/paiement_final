Get started

Register to get your api key

To get started with the Flitpay; there are just few steps that one should follow. First of all you should register to receive an api key. This api key will be later couple with your api script in order to identified and secure all your client transactions.

Download the script

  • Via the CDN 
    <script src="cdn://.../wearetechapi-v1.js" ></script >
  • On Github 
    <script src="github://.../wearetechapi-v1.js" ></script >
  • Or take it from our server. 
    <script src="https://paiementback.herokuapp.com/assets/js/wearetechapi-v1.js" ></script >

Set your script API Key

After your registration on our plateforme you receive an api key. Use our api method WAPI.setApiKey(< fill your api key >)

Define the call back method

The next step is to define the call back method WAPI.setApiCallback(function(data)). This method will help you to handle responses from your clients operation. The parameter < data > is an objet with describe the status of an initiated operation. For more details on this, see the documentation..

Create your buttons

Now to take on a payment, you just have to create a specific button in your page. This code below show the the template of a generic flitpay button where :
  • amount: is total amount the client whil have to pay.
  • Should-Script-Handle: is a value which describe whether or not our API should handle the button.
    • If this value is "0" the script will not handle click event on this button
    • If it's every value different from 0, the click even on this button will be handle by our API
 
 <button data-amount="<amount>" data-wearetechbtn="<Should-Script-Handle>" >Acheter cahier</button>
